## 1. Camera Movement Dictionary (44 Terms)

### 1A. Dolly Movements (Camera Physically Moves Forward/Backward)

| Term | Prompt Keyword | Description |
|------|---------------|-------------|
| Slow Dolly In | `camera moves slowly forward toward the subject` | Gradual approach, builds intimacy |
| Slow Dolly Out | `camera moves slowly backwards away from the subject` | Reveals context, creates distance |
| Fast Dolly In | `camera moves rapidly forward toward the subject` | Urgency, impact |
| Push In | `camera pushes in toward subject` | Emphasis, focus |
| Pull Back / Pull Out | `camera pulls back from subject` | Revelation, context |
| Vertigo Effect (Zolly) | `dolly zoom, camera moves backward while zooming in` | Background expands, disorientation (Hitchcock) |

### 1B. Pan & Tilt (Camera Rotates on Fixed Position)

| Term | Prompt Keyword | Description |
|------|---------------|-------------|
| Pan Left | `camera pans left` | Horizontal rotation left |
| Pan Right | `camera pans right` | Horizontal rotation right |
| Tilt Up | `camera tilts vertically upward` | Reveals height, power |
| Tilt Down | `camera tilts vertically downward` | Reveals ground, descent |
| Whip Pan | `camera whips violently to the side, motion blur` | Extreme speed, transition |
| Dutch Angle (Roll) | `camera tilted sideways on Z-axis` | Tension, unease |

### 1C. Truck / Lateral (Camera Moves Sideways)

| Term | Prompt Keyword | Description |
|------|---------------|-------------|
| Truck Left | `camera moves sideways on a track to the left` | Parallel lateral movement |
| Truck Right | `camera moves sideways on a track to the right` | Parallel lateral movement |
| Side Tracking (Parallel) | `camera trucks alongside the subject` | Walking with subject |

### 1D. Crane & Pedestal (Vertical Movement)

| Term | Prompt Keyword | Description |
|------|---------------|-------------|
| Crane Up | `camera lifts high into the air` | Epic reveal, ascension |
| Crane Down | `camera descends slowly to the subject` | Landing, approach |
| Pedestal Up | `camera rises vertically straight up` | Elevation, formality |
| Pedestal Down | `camera lowers vertically straight down` | Grounding, intimacy |

### 1E. Orbit / Arc (Camera Circles Subject)

| Term | Prompt Keyword | Description |
|------|---------------|-------------|
| Orbit 180 | `camera orbits 180 degrees around the subject` | Half-circle reveal |
| Orbit 360 | `camera orbits 360 degrees around the subject` | Full rotation |
| Fast 360 Orbit | `camera spins rapidly 360 degrees around subject` | Dynamic energy |
| Slow Cinematic Arc | `camera moves in a wide curve to reveal side profile` | Elegant reveal |
| Bullet Time | `frozen moment, ultra slow motion, camera orbit` | Time-freeze (Matrix) |

### 1F. Zoom (Lens Changes, Camera Static)

| Term | Prompt Keyword | Description |
|------|---------------|-------------|
| Smooth Optical Zoom In | `lens magnifies subject, camera stays stationary` | Focus attention |
| Smooth Optical Zoom Out | `lens widens, camera stays stationary` | Reveal context |
| Snap Zoom (Crash Zoom) | `rapid zoom directly into subject` | Shock, comedy |
| Cosmic Hyper Zoom | `fast zoom from extreme wide to macro level` | Scale transition |
| Extreme Macro Zoom | `zoom from subject to micro detail of surface` | Texture reveal |

### 1G. Tracking Shots (Camera Follows Subject)

| Term | Prompt Keyword | Description |
|------|---------------|-------------|
| Following Shot | `camera follows behind subject matching speed` | Pursuit, journey |
| Leading Shot | `camera moves backward matching subject's speed` | Face reveal while moving |
| Side Tracking | `camera trucks alongside the subject` | Parallel journey |

## 10. Constraint / Negative Prompt Dictionary

### 10A. Seedance-Specific Constraints (Pattern Blockers)

| Issue Category | Negative Prompt Terms |
|---------------|----------------------|
| **Motion Artifacts** | `no jitter, no warping, no shake, no flicker, no morphing` |
| **Identity Drift** | `no face changes, match identity exactly, keep same face and clothing` |
| **Style Drift** | `no color shift, no stylization changes, no added props` |
| **Text Issues** | `no garbled text, keep label perfectly readable, no text distortion` |
| **Object Issues** | `no logo morphing, no unwanted objects, no extra figures` |
| **Composition** | `locked horizon, subject stays centered, no camera shake` |

### 10B. General AI Video Negative Terms

| Category | Terms to Include in Negative Prompt |
|----------|-------------------------------------|
| **Quality** | `low quality, blurry, grainy, pixelated, low resolution, jpeg artifacts` |
| **Anatomy** | `deformed, extra limbs, extra fingers, missing arms, bad anatomy, fused fingers` |
| **Face** | `poorly drawn face, asymmetrical face, cloned face, distorted features` |
| **Hands** | `poorly drawn hands, broken wrist, extra fingers, missing fingers` |
| **Composition** | `out of frame, cropped, split image, disconnected limbs` |
| **Artifacts** | `watermark, text, logo, signature, username, banner` |
| **Style** | `oversaturated, undersaturated, monochrome (unless intended), kitsch` |
| **Motion** | `jerky motion, sudden speed changes, unnatural movement, teleporting` |

### 10C. Layered Negative Prompt Strategy

| Layer | Example |
|-------|---------|
| **Technical** | `no flicker, no blur, no compression artifacts` |

| **Motion** | `no jitter, no morphing, no sudden camera jumps` |
| **Continuity** | `no style drift, no color shift, no lighting change between shots` |
