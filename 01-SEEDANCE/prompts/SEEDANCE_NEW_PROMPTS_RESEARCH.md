### 3. "Seedance 2.0 Prompts Explained" - Oprelia AI (@oprelia-ai) [1.5K views]
- **DEDICATED PROMPTING GUIDE** - extremely valuable

**PROMPT STRUCTURE FORMULA (3 parts + optional 4th):**
1. **Opening Scene** - What you see first: "The scene starts with [image one] who is walking to the center of the room"
2. **Action** - What happens: "She snaps her fingers and her costume turns from gold to silver"
3. **Shot Description** - Technical details: "The video is in 4K, natural lighting"
4. **Audio Description** (optional): "We can hear sneaker sounds as she walks"

**Three Prompt Formats:**
1. **Standard format** - opening + action + description + audio
2. **ChatGPT-generated structured format** (using Noble Goose's prompt generator) - adds image tags, reference mapping, detailed scene description
3. **Second-based timeline format** - MOST PRECISE: "At this second this is happening, then at this second something different" - "Following the prompt most closely"

**Key Tips & Tricks:**
- Always use reference images (same credit cost as text-to-video)
- Keep character on blank/plain background for best reference
- Remove background from reference images for best results
- Up to 9 images, 3 videos, 3 audio clips as reference
- Video reference duration should be SHORTER than prompt duration
- Fast model is fine for simple scenes; Pro model for complex multi-action scenes
- **CENSORSHIP LOOPHOLE**: Translate violent prompts into Chinese Mandarin - "The western version has a lot more filters... Chinese is actually ironically less censored"
- **Language consistency**: Don't mix languages in dialogue - causes accent flips
- **End-of-prompt negatives**: Always add "no subtitles" and "no music" at the end
- "Generate video without subtitles" prevents auto-subtitles on dialogue
- "No music" prevents random music that won't match across clips
- All negative elements go at END of prompt
- Don't overload a 3-second segment with too many actions

**Consistency Techniques:**

- Generate close-ups so Seedance doesn't invent details at different shot sizes
- Plan for different lighting conditions (dark caves vs bright desert)
- Pre-generate specific variations you know you'll need (e.g., helmet open, helmet closed)
- If object doesn't behave right, generate additional reference showing the specific state (e.g., bike WITHOUT the case for "case falls off" scene)

**Collage Technique (when limited to 4 images):**
- Combine character images, wide shots, close-ups, and object variations into single collage images
- Flashboards platform offers automatic and manual collage creation
- Adjust collage structure for EACH SCENE based on what's needed

**Prompt Tips:**
- Use ChatGPT: provide script context, ask for 5-10 different shots within single scene
- Generate several 10-15 second variations, pick best moments for dynamic edit
- Prompt length limit: 1,990 characters (introduced due to high demand)
- Rephrase violence-related words in abstract ways to avoid filters
- Generate ALL shots without music (sound effects + voices only)
- Add music during editing, not generation
- Always upscale reference images to 4K with Nano Banana before using in Seedance
- Mark each image in prompt and reference them in shot description
- Now supports up to 9 images per prompt (recent update)

**Iterative Prompting Process (13 attempts documented):**
- Attempt 1-2: Storyboard + too many characters = mixed results
- Attempt 3: No storyboard, lots of camera switches, described enemies without reference images - "it clearly picked up enemies visible in the sword image"
- Attempt 4: Explicitly specified enemies (masked samurai)
- Attempt 5: Same prompt, different enemy types - proved Seedance understands prompts well
- Attempt 6: Started with close-up of sword dragging, creating sparks - mysterious opening
- Attempt 7: Fed storyboard but got single shot (no camera changes)
- Attempt 8: Explicitly specified different camera angles - problem was not properly referencing close-up face image
- Attempt 9: Three reference images for consistency
- Attempts 10-12: Continued refinement
- Attempt 13: FINAL WINNING STRUCTURE

**The Winning 4-Part Reference Structure:**
1. Characters in same style (all realistic OR all stylized - never mix)
2. Characters uploaded separately (each main character as own reference)
3. Key objects and locations as references (sword, cave, etc.)
4. Storyboard with action frames showing narrative

**Director Tips:**
- Describe different camera angles DIRECTLY in the prompt
- Describe choreography in detail - "samurai laughing, looks left and right at others, then suddenly launches forward at high speed"
- Seedance works really well with emotions and choreography
- For realistic results, explicitly describe behavior
- Use Pinterest for composition, lighting, fight angle references
- Start simple, then layer complexity
- First try is rarely perfect - generate 3 variations at once
- Write "cinematic quality" literally in prompts - produces strong, dramatic results
- Compare variations side by side using slideshow mode

## COMPILED MASTER TECHNIQUE LIST

### PROMPT STRUCTURE FORMULA
1. **Opening Scene**: What the viewer sees first
2. **Action**: What happens in the scene
3. **Shot/Technical Description**: 4K, lighting, weather, camera
4. **Audio** (optional): Sound effects, footsteps, ambient
5. **Negative prompts at END**: "No subtitles, no music, no dialogue, no text on screen"

### THREE PROMPT FORMAT OPTIONS
1. **Standard**: Opening + Action + Description (good for simple scenes)
2. **ChatGPT/Claude structured**: Full scene breakdown with image tags (good for complex scenes)
3. **Second-by-second timeline**: "0-3s: X happens. 3-6s: Y happens." (MOST PRECISE control)

### REFERENCE IMAGE BEST PRACTICES
- Use plain/white backgrounds for character references
- Generate characters from MULTIPLE ANGLES
- Include close-ups for face details
- Generate different lighting conditions (dark/bright)
- Pre-generate specific states you'll need (helmet on/off, object attached/detached)
- Upscale all references to 4K before using in Seedance
- Create collages when image slots are limited
- Style consistency: ALL references must match (all realistic OR all stylized)

### PRODUCTION WORKFLOW
1. Write script with 3-act structure
2. Generate character references (multiple angles, close-ups)
3. Generate object/location references
4. Create storyboard frames
5. Upscale everything to 4K
6. Generate with NO music/voiceover (sound effects only)
7. Generate separately WITH voiceover for audio layer
8. Stitch best moments from multiple generations
9. Add music in post-production

### KEY TOOLS TO USE WITH SEEDANCE
- **Google Gemini**: Analyze YouTube videos for prompt reverse-engineering; generate prompts from reference videos
- **Claude**: Character sheets, prompt writing, script generation (preferred by professionals)
- **ChatGPT**: Scene descriptions, structured prompts
- **Nano Banana Pro**: Image generation, upscaling to 4K, character multi-angle shots
- **Flashboards / Martini Art**: Project management, batch generation, collage creation

### CENSORSHIP WORKAROUNDS
- Translate violent/mature prompts into Chinese Mandarin for fewer filters
- Rephrase violence-related words abstractly
- Some false flags occur - retry same prompt if unexpectedly blocked
- "No subtitles" prevents unwanted auto-subtitles

### COMMON MISTAKES TO AVOID
- Mixing realistic + stylized references in same generation
- Overloading 3-second segments with too many actions
- Forgetting to specify "no music" (causes audio mismatches across clips)
- Not describing each input's role to the model
- Using low-res reference images (always upscale to 4K)
- Trying to generate readable text on screen (gibberish results)
- Mixing languages in dialogue (causes accent contamination)
- Not planning which reference images you need before starting
