### What ControlFreak Does
- Maps **any physical MIDI controller** (knobs, pads, sliders, pitch wheel) or **gamepad** (buttons, analog sticks, triggers) to ComfyUI parameters
- Can control: node sliders, number inputs, toggles, dropdowns, and UI commands (Queue Prompt, Interrupt, etc.)
- Auto-detects connected devices
- Mappings persist within workflow JSON (shareable!)
- Visual feedback with orange glow on mapped widgets
- Profile saving/loading for different workflows
- Quick Map via right-click context menu
