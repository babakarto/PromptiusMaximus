# Part 2 — Meta Video Ad Script
## Candy.ai | 20s Feed/Story Ad | SFW Adaptation

---

### Strategic Approach

**Core Insight:** The same users browsing Pornhub are scrolling Instagram and TikTok. By using the **same AI-generated faces** across both platforms, we create subconscious brand recall — "Wait, isn't that the girl from...?" This bridges the gap between NSFW and SFW advertising, turning platform restrictions into a retention advantage.

**Format:** UGC-style, direct-to-camera. The AI girl speaks naturally, like a TikTok creator — relatable, casual, slightly provocative but fully Meta-compliant. Each variant targets a specific fantasy archetype without showing anything explicit.

**Destination:** candy.ai/characters/new

### Cross-Platform Recall Strategy

```
PORNHUB (Explicit)          META (Soft/SFW)
    |                           |
    +--- SAME AI FACE ----------+
    |                           |
 Pre-roll ad               UGC-style reel
 "Create your girlfriend"  "Build your dream girl"
 Explicit visual hook       Curiosity + suggestion
    |                           |
    +---> candy.ai/characters/new
```

The user sees the same girl in two contexts. On Pornhub, she's in an explicit ad. On Instagram, she's talking to them like a creator. This creates:
1. **Recognition** — "I've seen her before"
2. **Curiosity** — "She's on Instagram too?"
3. **Trust** — repeated exposure = familiarity
4. **Conversion** — multiple touchpoints drive higher intent

---

### A/B Testing Plan
- **Hook:** "You've been scrolling..." vs "Be honest..." vs "POV: she's exactly your type"
- **CTA:** "Create yours" vs "Build her" vs "She's waiting"
- **Face consistency:** Same girl across all variants vs different girls per variant
- **Format:** Vertical 9:16 story vs square 1:1 feed

### Meta Compliance Notes
- No nudity, no explicit language, no sexual acts
- Positioning: "AI companion" / "virtual girlfriend" — not "AI porn"
- All visuals are AI-generated (no real humans)
- Suggestive tone only — curiosity-driven, never graphic
