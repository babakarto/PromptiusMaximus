# CapCut Keyboard Shortcuts — Complete Reference

> Extracted from all 5 Patrik Key tutorials. Your cheat sheet for fast editing.

---

## Core Editing

| Shortcut | Action | Context |
|----------|--------|---------|
| **S** | Split all layers at playhead | Use constantly for cuts — works on ALL layers at once |
| **Q** | Delete everything to the LEFT of playhead | Fastest way to trim. Replaces B > X > A (3 clicks → 1) |
| **W** | Delete everything to the RIGHT of playhead | Same as Q but opposite direction |
| **X** | Delete selected clip | Remap from Backspace — keeps your left hand on the keyboard |
| **A** | Deselect cutting mode | Use after cutting to go back to selection mode |
| **B** | Activate cut/blade tool | Default cut tool (but Q/W are faster for most cases) |

---

## Layer & Visibility

| Shortcut | Action | Context |
|----------|--------|---------|
| **V** | Hide/Show (disable/enable) layer | Use to organize captions, hide reference text layers, toggle visibility |
| **Command + Click** (Mac) / **Alt + Click** (Win) | Deselect individual layer from multi-selection | Hold while clicking to exclude one layer from a group selection |

---

## Markers & Organization

| Shortcut | Action | Context |
|----------|--------|---------|
| **M** | Add marker at playhead position | Mark the end of each reel in batch editing, label sections |

---

## Export & I/O Points

| Shortcut | Action | Context |
|----------|--------|---------|
| **I** | Set IN point (export start) | Place playhead at start of reel → press I |
| **O** | Set OUT point (export end) | Place playhead at end of reel → press O, then Export |

---

## Timeline Navigation

| Shortcut | Action | Context |
|----------|--------|---------|
| **Right Arrow** | Move 1 frame forward | Precise frame-by-frame navigation (e.g., 5 frames for counter effect) |
| **Left Arrow** | Move 1 frame backward | Precise frame-by-frame navigation |
| **Shift + Right Arrow** | Move 10 frames forward | Fast jump for setting up animation keyframes |
| **Shift + Left Arrow** | Move 10 frames backward | Fast jump backward |

---

## Animation & Keyframes

| Shortcut | Action | Context |
|----------|--------|---------|
| **Shift + W** | Smooth selected keyframes automatically | Select keyframes first, then Shift+W — replaces manual graph editing |

---

## Compound Clips

| Shortcut | Action | Context |
|----------|--------|---------|
| **Option + G** (Mac) | Create Compound Clip | Essential for unlocking: masks, glow, freeze, advanced effects |
| **Right-click > Create Compound** | Same as above (all platforms) | Use when shortcut isn't available |

## Speed Control

| Action | How | Context |
|--------|-----|---------|
| Change clip speed to 2x | Select clip > Speed > set to 2 | Sprint through footage in batch editing |
| Reset clip speed to 1x | Select clip > Speed > set to 1 | Reset after raw cutting each reel |

## Pro Workflow Combos

### Fast Raw Cut (Batch Editing)
```
Q → Q → Q → (move playhead) → Q → ...
```
Sprint through footage deleting everything to the left. Way faster than B > X > A.

### Full Reel Trim & Label
```
S (split) → Q or W (delete) → ... repeat ...
→ M (marker at end)
→ drag text layer → type label → V (hide label)
```

### Export One Reel from Batch Timeline
```
(playhead at start) → I (in point)
(playhead at end/marker) → O (out point)
→ Export → paste name → done
```

### Caption Animation Setup
```
V (hide layer) → position layer
→ keyframe Transform + Blend
→ Shift + Right Arrow x10 (10 frames forward)
→ keyframe again
→ go back to first keyframe → set values
→ select both keyframes → Shift + W (smooth)
```

### Left-Hand-Only Editing (never leave the mouse)
```
A (deselect) → S (split) → Q (delete left) → X (delete) → W (delete right)
All keys are on the left side of the keyboard.
```

## Variable Speed Animation — Not a Shortcut, But Essential

Right-click any clip > **Show Variable Speed Animation** to access:

| Option | Effect |
|--------|--------|
| **Auto Curve** | Automatic ease in/out (select ONE keyframe at a time) |
| **Quad Out** | Fast start → slow end (select multiple keyframes) |
| **Rebound Out** | Bounce/overshoot effect (great for shine animations) |
| **Blue line drag** | Manual control: drag toward front = fast start, slow end |

> **Rule:** For Auto Curve, always select only ONE keyframe. If you select both, the button won't work.
