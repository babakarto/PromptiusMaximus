# Edit 30 Reels in 4 Hours (Batch Editing System)

> **Author:** Patrik Key
> **Video:** https://www.youtube.com/watch?v=Ksw3MtE7lg0
> **Key takeaway:** A system to batch edit 30 reels in one 4-hour session using CapCut

---

## Phase 1: Raw Cut — Sprint Through the Footage

### Initial Setup
1. Select **all your reels** and put them on **one single timeline**
2. Go to **Speed** and bring it up to **2x** (sprint through the footage faster)
3. Don't worry, you can always reduce it when you're done editing

### Shortcuts for Raw Cutting

| Key | Action |
|-----|--------|
| **Q** | Deletes everything to the left |
| **W** | Deletes everything to the right |
| **S** | Split (cut) |

### What to Focus On
- Remove **silences** (look at the audio waves — where they're flat, nothing is going on)
- Remove **bad takes**
- Don't worry about details, just cut the fat

### After Each Reel — Organization
1. **Select the clips** of the reel you just edited
2. Go to **Speed** > reduce back to **1x**
3. Press **M** = add a **Marker** at the end of the reel
4. Drag a **default text layer** and type the topic/title of the video
5. Press **V** to disable the text layer (it's just for reference)

> Result: 12 reels raw-cut in **20 minutes**. With 30 videos, roughly **1 hour**.

---

## Phase 2: Create the Text Hook

The text hook appears at the beginning of the reel to tell viewers what the video is about.

### How to Build the Text Hook
1. Choose your **font** (e.g., Poppins)
2. Type the first word of the hook (e.g., "Collect")
3. Apply **pop in** animation
4. Make it bigger
5. **Duplicate the layer** and type the second word (e.g., "Payment")
6. Change the font on the second word (e.g., Shell Edit — it's trending)
7. Adjust sizes so everything fits

### Replicate the Text Hook
1. Create the text hook for the first reel
2. Go to the beginning of the **next reel**
3. **Paste** the text hook
4. Change only the text
5. Repeat for all reels

> Play with colors if you want (e.g., keyword in yellow to make it pop).

---

## Phase 3: Export with I/O Points

### Export Shortcuts

| Key | Action |
|-----|--------|
| **I** | Set the **In point** (start) |
| **O** | Set the **Out point** (end) |

### Procedure
1. Place playhead at the start of the reel > press **I**
2. Go to the end of the reel (where the marker is) > press **O**
3. Click **Export**
4. Copy the name from the text layer and paste it as the filename
5. Export
6. Repeat for every reel

---

## Phase 4: AI Captions (Submagic)

After exporting all reels:

1. **Upload** the video to Submagic
2. Let it **generate captions** automatically
3. Choose a **caption style** (e.g., Eman style)
4. Customize: font, size, color
5. **Save the style** so you don't have to recreate it every time
6. Optional: add music, sound effects, b-roll from the platform

---

## Phase 5: Scheduling & Publishing

Directly from Submagic:
1. Go to the **Publishing** section
2. Click **Publish**
3. Select the **reels batch**
4. Write/edit the **description** (auto-generated by AI)
5. Choose platform: Reel, Story, YouTube Shorts
6. Optional: add a **first comment**
7. **Schedule**: select timezone, date, and time
8. Hit Schedule and the video is done

> Upload all videos to Submagic, make all captions, schedule them, and walk away from your computer for the rest of the month.

| **TOTAL** | **~4 hours** |

## The System Summary

```
1. Import EVERYTHING on one single timeline
2. Speed up to 2x to sprint
3. Q/W/S for super fast raw cut
4. Marker + Text label for each reel
5. Reduce speed back to 1x after cutting
6. Create 1 text hook > replicate across all reels
7. Export with I/O points
8. AI captions on Submagic
9. Schedule everything > go grab a coffee
```
