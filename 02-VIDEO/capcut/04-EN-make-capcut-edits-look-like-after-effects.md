## 1. Slide Up/Down Caption Animations

The cleanest, most premium looking animation — After Effects style slide up/slide down captions.

### Preparing the Captions:
1. Select your clip > go to **Captions** > **Generate captions**
2. Select whichever **font** you like
3. Go to the **Captions** section
4. **Split every single word manually**: click between two words and hit **Enter**
5. Repeat for all words

### Positioning the Captions:
1. **Select all** captions
2. On Mac: hold **Command** and click the first layer (deactivates it)
3. Move all layers above, creating a **2-frame gap** between each layer
4. On Mac: hold **Command** + click first layer again, continue for the rest
5. Select all > drag up
6. Position playhead > press **S** to cut > delete the excess
7. Select all captions > press **V** to disable them

### Visual Positioning:
1. Activate each layer one by one with **V**
2. Use the **guide line** so everything is on the same level
3. Position each word right next to the previous one
4. Select all text layers > position wherever you like (e.g., bottom)
5. Resize if needed

### Applying the Animation:
- **Slide Up:** Go to **Animation** > look for **"Slide Up"** > add to **Favorites**
- **Slide Down:** Go to **Animation** > look for **"Slide Down"** > add to **Favorites**

### Bonus — Out Animation:
1. Go to **Animation** > **Out**
2. Select **Blur**
3. Set speed to **0.6 seconds**
4. When you activate the second layer, it creates a cool transition effect

> One effect in and your edit already looks like it belongs in After Effects.

## 2. Custom Backgrounds from Scratch

Stop searching for backgrounds all over the internet. Build your own with any color and any texture, all inside CapCut.

### Building the Base:
1. Go to **Text** > add **Default Text**
2. Make the layer longer
3. Remove the default text
4. Go to **Font Size** > **max it out**
5. Scroll down to **Background** > pick your **first color** (e.g., blue)
6. **Duplicate** the layer
7. On the duplicate: change **Background** to your **second color** (e.g., black)
8. Create **separate Compound Clips** for each layer

### Creating the Gradient:
1. Select the **dark layer** > **Video** > set it on **Layer 2**
2. Go to **Mask**

**Option A — Split:**
- Select **Split** > rotate > play with the **Feather**
- Result: dark color on the bottom, lighter on top (or vice versa)

**Option B — Circle:**
- Select **Circle** > click **Reverse** > drag it up
- Play with the **Feather**
- Result: circular gradient

### Adding Texture:

**Method 1 — TV Lines:**
1. Select both backgrounds > **Create Compound**
2. Go to **Effects** > search **"TV Lines"**
3. Drag it on the compound clip
4. Decrease the **Range** for subtle texture

**Method 2 — Media Assets:**
1. Drag any asset onto the timeline
2. Find the frame you like > use **Freeze**
3. Delete the parts before and after
4. Go to **Video** > **Basic** > **Blend** > set to **Screen**
5. Decrease the **Opacity**
6. Rotate and position as you like

### Changing Background Color:
- Double-click the light layer > select the text layer > change color in Background
- On the dark layer: go to **Mask** and adjust

### Adding Motion:
1. Go to **Effects** > search **"Play Pendulum"**
2. Drag it on the asset layer
3. The background now moves!

### Advanced Color Tweaking:
- Go to **Adjustments**: Temperature, Exposure, Contrast
- Go to **Curves** for total color control

## 3. Custom Shapes + Color Morphing

Create any shape you want and smoothly morph its color — like an actual motion graphics project, all inside CapCut.

### Color Morphing:
1. **Text** > Default Text > remove text > max font size > **Background** > first color (e.g., green)
2. Click the **keyframe** on Background
3. Go **1 frame forward**
4. Change color (e.g., yellow/orange)
5. Click **keyframe** again

> **Important:** The gap between keyframes must be as small as possible (1 frame) for an immediate transition. Bigger gaps = colors mixing together.

### Custom Shapes with Mask:
1. Right-click the text layer > **Create Compound Clip**
2. **Video** > **Mask** > **Rectangle**
3. For a circle: round corners to the max
4. Set **keyframe** on the position
5. Move a few frames forward > drag the shape (e.g., to the left)
6. To change from circle to rectangle:
   - Add keyframe > move forward
   - Reduce **Round Corners** from 100 to 15
   - Change size as you like

### Smoothing the Animations:
1. Right-click > **Show Variable Speed Animations**
2. Select all keyframes > **Quad Out**
3. For individual keyframes: select only one > **Auto Curve**

> **Note:** For Auto Curve, select **only one** keyframe at a time. If you select both, you won't be able to click Auto Curve.

### Adding Stroke (Edges) to Shapes:
1. **Duplicate** the shape layer
2. **Video** > **Basic** > make it slightly bigger
3. **Adjustment** > **Curves** > bring the line all the way up
4. **Video** > set on **Layer 3**
5. Play with the **Mask** for the border

### Bonus — Glow on Shapes:
- Go to **Effects** > search **"Glow Too"** > drag it on the shape

## 4. Scene Transitions (After Effects Style)

Most editors just do roll cuts or use boring transitions everyone else is using. Here's how to transition from one scene to another — super simple, super smooth.

### Film Strip Transition:
1. Build the **destination background** (e.g., white with black gradient, see section 2)
2. Select both layers > **Create Compound**
3. Place playhead where you want the transition
4. Go to **Mask** > **Film Strip** > make it small
5. Set **keyframe**
6. Go a few frames forward > drag the mask to cover the screen
7. **Smooth:** right-click > Variable Speed Animation > Auto Curve on both keyframes
8. Drag the **blue line** for: fast at the start > slows down at the end

### Bouncing Ball Transition:
1. Create the background (e.g., purple)
2. **Mask** > **Rectangle** > corners at **100** (circle)
3. Set the initial **keyframe**
4. Move playhead > position the ball > repeat (create the bounce)
5. For the final transition: drag the edge of the ball upward to cover the screen
6. **Smooth** with Variable Speed Animation > Auto Curve on every keyframe

> **Note:** Time the transition exactly with your text so everything is seamless.

## 5. Deep Glow V2 (Without Paying $99)

The famous After Effects plugin that costs $99... free in CapCut. Even an After Effects editor said the glow is very similar.

### How To:
1. Create your text
2. **Right-click** > **Create Compound Clip** (mandatory — you can't apply glow directly on text)
3. Go to **Effects** > search **"Glow"**
4. You'll see: **Edge Glow** and **Glow Too**
5. Add to **Favorites** for quick access
6. Drag **Glow Too** onto the text compound
7. Play with the settings

> **Works on:** text, icons, shapes, anything you want

### Important Notes:
- **Don't overuse it** — your viewers don't need sunglasses when watching your video
- If you don't see the functions: go to **Settings** and make sure you're on **version 6.40+**

## Key Concepts Summary

| Concept | What It Does |
|---------|-------------|
| **Compound Clip** | The key to unlocking advanced effects (mask, glow, freeze) |
| **Variable Speed Animation** | Smooths out any keyframe animation |
| **Auto Curve / Quad Out** | The two most-used easing options |
| **Blue line in keyframes** | Controls acceleration/deceleration |
| **Mask** | The most powerful tool in CapCut (split, circle, film strip, rectangle) |
| **Background on Text Layer** | The method to create custom shapes and backgrounds |
